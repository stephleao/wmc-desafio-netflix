## ⚙️ Como rodar o projeto

1. Clone o repositório

2. Instale as dependências:

  ``` bash
  pip install -r _dependencias.txt

  # OU

  pip install pandas jupyter ipykernel missingno
  ```

3. Execute o Notebook:

- Abra `Nina_da_Hora...2026_1.ipynb` no VS Code
- Selecione o kernel no canto superior direito
- Execute tudo ou célula por célula

## 🚀 Tecnologias Utilizadas

- Python 3.x
- Pandas — manipulação e análise de dados
- Jupyter Notebook / Google Colab — ambiente interativo
- VS Code + extensões Python e Jupyter
